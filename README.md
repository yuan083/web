# 税务师学习平台 - 本地版

一个功能完整的税务师考试学习系统，支持多用户、多科目、间隔重复学习。

## 🚀 快速开始

### 1. 启动应用
由于使用了IndexedDB，需要通过HTTP服务器访问：

```bash
# 方法1: 使用Python
python3 -m http.server 8000

# 方法2: 使用Node.js
npx serve

# 方法3: 使用VS Code Live Server插件
# 右键点击index.html，选择"Open with Live Server"
```

### 2. 访问应用
在浏览器中打开：`http://localhost:8000`

## 🎯 使用流程

### 1. 注册账户
- 打开应用后点击"注册"
- 填写邮箱和密码
- 完成注册后自动登录

### 2. 创建科目
- 点击"科目"菜单
- 点击"新建科目"
- 填写科目信息并选择可见性

### 3. 开始学习
- 在科目列表中选择要学习的科目
- 点击"开始学习"
- 使用间隔重复算法进行高效学习

### 4. 查看统计
- 在首页查看学习统计
- 在个人资料页面管理账户设置

## 📊 学习算法

### 间隔重复系统
- **等级0**: 1天后复习
- **等级1**: 3天后复习
- **等级2**: 7天后复习
- **等级3**: 14天后复习
- **等级4**: 30天后复习
- **等级5**: 60天后复习
- **等级6**: 120天后复习
- **等级7**: 240天后复习

### 记忆规则
- **回答正确** → 升级等级，延长复习间隔
- **回答错误** → 重置到等级0，第二天重新复习

## 📝 开发说明

### 数据库架构
应用使用IndexedDB存储8个主要数据表：
- `users` - 用户信息
- `decks` - 科目数据
- `cards` - 卡片内容
- `progress` - 学习进度
- `study_sessions` - 学习会话
- `user_stats` - 用户统计
- `bookmarks` - 书签
- `media_files` - 媒体文件
